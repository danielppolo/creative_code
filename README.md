# Generative Design
Projects from the book.

### New project

```
$ cd my_collection
$ p5 generate my_project
# or... 
$ p5 g my_project
```

With ES6
```
$ p5 g my_project_es6 --es6
```

### Server

```
$ p5 server
# or... 
$ p5 s
```

Customize port
```
p5 server --port 8888
# or... 
p5 s -p 8888
```
